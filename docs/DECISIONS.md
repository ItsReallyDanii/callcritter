| Browser-first MVP | Locked | Fastest route to playable contest link |

| Server-side rate limit | Pending before broad public launch | Needed for uncontrolled traffic |
| Manual placement over auto-placement | Locked | GPT-5.5 recommends placement; the user controls the final visual |
| Share-card export | Implemented | Creates the contest-friendly artifact without adding storage or accounts |
